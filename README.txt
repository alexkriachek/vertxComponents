
1) Find places endpoint:
Method - GET
http://localhost:8080/?country={country}&user_text={user_text}, where
{country} - country code,
{user_text} - user text.
for example:
http://localhost:8080/?country=ua&user_text=Kyiv

2) Get metrics endpoint:
Method - GET
http://localhost:8081

3) I used SoapUI application and browser for testing.
